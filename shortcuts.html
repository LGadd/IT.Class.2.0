Common Keyboard Shortcuts

Action Shortcut
Copy Ctrl + C (Windows) / Cmd + C (Mac)
Paste Ctrl + V (Windows) / Cmd + V (Mac)
Cut Ctrl + X (Windows) / Cmd + X (Mac)
Select All Ctrl + A (Windows) / Cmd + A (Mac)
Undo Ctrl + Z (Windows) / Cmd + Z (Mac)
Redo Ctrl + Y (Windows) / Cmd + Shift + Z (Mac)
Save Ctrl + S (Windows) / Cmd + S (Mac)
Find Ctrl + F (Windows) / Cmd + F (Mac)

Tips for Efficient Work

  • Learn shortcuts for the software you use frequently—these can significantly save time.
  • Practice using shortcuts regularly until they become second nature.
  • Explore customization options in your tools to set your own shortcuts.
